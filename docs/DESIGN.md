# Design System Documentation

## Theme Overview

This document outlines the core design principles and token values that define the aesthetic and functional characteristics of our user interface.

## Color Palette

Our color palette is designed to create a clear visual hierarchy and a consistent brand experience. The current theme operates in **light mode**.

- **Primary Color:** `#A5C7E9` - This is our brand's most distinctive color, used for key interactive elements, buttons, and calls to action.
- **Secondary Color:** `#78909C` - A supporting color, suitable for less prominent UI elements, chips, and secondary actions.
- **Tertiary Color:** `#B0BEC5` - An additional accent color for highlights, badges, or decorative elements.
- **Neutral Color:** `#889ec0` - This serves as a neutral base for backgrounds, surfaces, and non-chromatic elements, providing a balanced visual foundation.

## Typography

Our typography system ensures readability and visual appeal across all content. All fonts are set to `epilogue`.

- **Headline Font:** `epilogue` - Used for all primary headings and titles, ensuring strong visual impact.
- **Body Font:** `epilogue` - Applied to paragraph text and general content for optimal readability.
- **Label Font:** `epilogue` - Utilized for labels, captions, and smaller text elements, maintaining consistency with the overall typographic style.

## Shape and Form

The user interface features a **subtle roundedness** (roundedness value of `1`). This choice helps create a modern and approachable feel, softening the edges of UI components without being overly playful.

## Spacing

The design adheres to **normal spacing** (spacing value of `2`), providing a balanced amount of whitespace around and within UI elements. This ensures a clean layout, prevents visual clutter, and enhances readability without feeling too dense or overly spacious.
